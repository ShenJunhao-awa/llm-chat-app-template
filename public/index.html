📖 接入指南

API 地址

POST /api/judge

请求格式

{ "content": "要审核的帖子内容" }

响应格式

✅ 通过：{ "status": "pass" } ❌ 拒绝：{ "status": "reject", "reason": "人身攻击", "violation_phrase": "蠢货、滚出去" } ⚠️ 异常：{ "status": "pass", "error": "审核服务异常，已默认放行" }

审核原则

• 骂产品/游戏/公司 → 放行 • 骂论坛里的其他用户 → 拦截 • 玩梗、网络用语、吐槽 → 放行 • 不确定 → 放行

前端集成代码

const JUDGE_API = 'https://llm-chat-app-template.jgp.dpdns.org/api/judge'; async function reviewPost(content) { const res = await fetch(JUDGE_API, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ content }) }); const data = await res.json(); if (data.status === 'reject') { alert('违规：' + data.reason + '\\n违规词：' + data.violation_phrase); return false; } return true; }
