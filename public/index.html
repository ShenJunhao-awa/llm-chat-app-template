📖 接入指南

API 地址

POST /api/judge

请求格式

{ "content": "要审核的帖子内容" }

成功响应

{ "status": "pass" } 或 { "status": "reject" }

异常响应

{ "status": "pass", "error": "审核服务异常，已默认放行" }

论坛集成代码（前端 CreatePost 页面）

const JUDGE_API = 'https://llm-chat-app-template.jgp.dpdns.org/api/judge'; async function reviewPost(content) { const res = await fetch(JUDGE_API, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ content }) }); const data = await res.json(); return data.status === 'pass'; }
